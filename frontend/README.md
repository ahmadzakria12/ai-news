# AI News Frontend

Modern Next.js frontend for AI News website with animations and AI agent integration.

## Features

- 🎨 Modern, animated UI based on Figma design
- 🤖 Integration with multiple AI agents (SEO, YouTube, Forbes, Web Search)
- 📱 Responsive design
- ⚡ Fast performance with Next.js
- 🎭 Smooth animations with Framer Motion
- 🔍 SEO optimized with meta tags

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env.local` file:
```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Build

```bash
npm run build
npm start
```
